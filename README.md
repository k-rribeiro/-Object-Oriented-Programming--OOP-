📘 Object-Oriented Programming (OOP)

This repository contains assignments and exercises developed during the Object-Oriented Programming course at the University of São Paulo (USP), São Carlos campus, at the Institute of Mathematical and Computer Sciences (ICMC). All implementations are written in Java.

📚 About the Course

The course introduces the fundamental principles of object-oriented programming and software development, emphasizing both theoretical concepts and practical applications.

Key topics include:

Encapsulation and information hiding
Separation of interface and implementation
Classes and inheritance
Polymorphism
Class hierarchies and design
Collections and iteration mechanisms
🎯 Purpose of this Repository

This repository aims to:

Document the learning process throughout the course
Provide clear and practical examples of OOP concepts
Serve as a reference for future studies and projects in Java
